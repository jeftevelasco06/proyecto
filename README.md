# Lista de Tareas con Tkinter

## Descripción

Esta es una aplicación sencilla de Lista de Tareas desarrollada en Python usando la biblioteca Tkinter para la interfaz gráfica. Permite a los usuarios añadir, eliminar y gestionar tareas de manera interactiva y fácil de usar.

---

## Funcionalidades

- Añadir tareas escribiendo en el campo de texto y presionando el botón "Agregar Tarea".
- Seleccionar una tarea de la lista y eliminarla con el botón "Eliminar Selección".
- Interfaz responsive que se adapta al tamaño de la ventana.
- Mensajes de advertencia si se intenta añadir una tarea vacía o eliminar sin seleccionar.

---

## Requisitos

- Python 3.x
- Tkinter (viene instalado por defecto en la mayoría de distribuciones de Python)
